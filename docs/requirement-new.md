- ปรับแก้กระบวนการ DAR ผู้ reviewer approver mr qms สามารถแก้ไขเอกสารที่ ผู้ร้องขออัปโหลดได้เพื่อให้ได้ฉบับที่สมบูรณ์ที่สุด.
- Export PDF ส่วนการนำจ่ายเอกสาร ตรง Department ให้เป็นชื่อตัวย่อแผนกที่ถูกตั้งไว้ qms/department-codes
- เมื่อผ่านกระบวนการสุดท้าย QMS อนุมัติเรียบร้อย เริ่มกระบวนการ แปลงไฟล์เป็น PDF พร้อม stamp ตราบริษัท อ้างอิง Folder public/stamp โดยแผนก QMS เลือก ขยับ กำหนดตำแหน่ง เสร็จแล้วกดคลิป publish ไปที่หน้าแจกจ่าย
- เพิ่มหน้าการแจกจ่าย ตั้งให้เลือกแผนกที่ต่้องมารับทราบ หากแผนกนั้นมีการ download แล้วให้บันทึกชื่อแผนกวันที่ Download ไว้ แล้วใน stamp ก็จะอัปเดตวันที่ตรง Date เป็นวันที่เขา download และ ข้างล่าง copy to จะเป็นชื่อแผนกที่กำหนดตัวย่อไว้ qms/department-codes 
- แล้วแผนกนั้นก็จะไม่สามารถ download ได้อีก  limit ได้แค่ แผนกละ 1 ครั้ง
- การแจกจ่าย จะเชื่อมต่อกับ หน้าเอกสารทั้งหมด qms/document-controls เอกสารที่อัปโหลดได้แจกจ่ายจะสามารถ Link ไปอัปเดตหน้าเอกสารทั้งหมดได้โดยมีปุ่มคลิกติกเลือกไว้ว่าต้องการให้ลิ้งหรือไม่ Default จะเป็น true เลือกไว้
- เอกสารหน้า qms/document-controls ที่ link กับการแจกจ่าย จะไม่มี stamp เหมือนหน้า แจกจ่าย

## สถานะการพัฒนา

- [x] กดแจกจ่ายแล้วแปลงไฟล์ต้นฉบับเป็น PDF (ไฟล์ที่ไม่ใช่ PDF ใช้ PDF rendition จาก SharePoint)
