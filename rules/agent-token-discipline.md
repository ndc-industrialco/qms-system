# Agent Token Discipline

* Do not read every rule file.
* Do not read every domain document.
* Do not load entire task logs.
* Read only recent relevant task-log entries.
* Prefer search over full-file reads.
* Open only files directly related to the task.
* Avoid repository-wide scans unless required.
* Mark unknown information as `UNKNOWN`.
* Do not invent architecture, APIs, schemas, or workflows.
