# Tech Stack

- Next.js 15 (App Router): Main framework for server-rendered pages, routing, and API route handlers.
- React 19: Frontend UI library for building interactive components.
- TypeScript: Primary language for strong typing and safer refactoring.
- PostgreSQL: Primary relational database for business data.
- Prisma ORM: Data modeling, query access, and migration workflow for PostgreSQL.
- NextAuth.js v5 (beta): Authentication and session management (Microsoft Entra ID provider).
- Tailwind CSS v4: Utility-first styling system for consistent UI development.
- Radix UI Primitives: Accessible base components (Dialog, Dropdown, Select, etc.).
- TanStack Query: Client-side server-state fetching, caching, and synchronization.
- React Hook Form: Form state management and submission handling.
- Zod: Runtime schema validation for API inputs and form payloads.
- Redis (ioredis): Caching and rate-limit/session-related controls.
- Microsoft Graph API: Integration for Entra users/groups, email, and SharePoint file workflows.
- Docker: Containerized build/runtime for deployment.
